# AboRadar - Landing Page

Marketing website for AboRadar, hosted on GitHub Pages.

**Live:** [https://dragethy.github.io/aboradar](https://dragethy.github.io/aboradar)

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Landing page with hero, features overview, pricing |
| `features.html` | Detailed feature list (available + coming soon) |
| `changelog.html` | Version history and release notes |
| `marketing.html` | Marketing plan, roadmap, KPIs |
| `datenschutz.html` | Privacy policy (DSGVO) |
| `impressum.html` | Legal notice |

## Tech

- Static HTML + Tailwind CSS (CDN)
- Dark theme matching the app design
- Responsive layout
- SEO optimized with sitemap.xml and robots.txt

## Design System

- **Background:** #08090D
- **Accent:** #00E89D
- **Cards:** #111318
- **Fonts:** Outfit (headings), DM Sans (body), JetBrains Mono (code)

## Deployment

Push to `main` branch - GitHub Pages auto-deploys from root.
